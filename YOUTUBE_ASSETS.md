### How it works:
Fractal uses a "Self-Improving Loop" that treats prompt engineering as an optimization problem. Every time it generates questions, an LLM Judge scores the quality of the output. Using W&B Weave traces, the system automatically updates its probability weights to favor the prompt strategies that yield the most insightful questions over time.
